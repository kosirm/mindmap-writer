# Settings

Settings are a set of user configurable options which can be accessed from the settings panel. Settings panel will be accessible from the main menu, from the command palette and by keyboard shortcut. 
- Settings panel will have two tabs - one for settings and other for keyboard shortcuts manager.
- Settings will be saved to localStorage for rapid access and can be saved also to google drive (synced across devices) automatically when user changes any setting. Settings file on google drive will be named _mindscribble_settings.json and will be created in the root of the app folder.
